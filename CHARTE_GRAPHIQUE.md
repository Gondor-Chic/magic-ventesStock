## 📏 Règles d'Usage

1. **Couleur dominante** : `#CC7722` doit être présente sur chaque élément (runes, bordures, labels, boutons, accents, séparateurs, vignette, textures)
2. **Palette terre ocre** : Privilégier `#CC7722` et ses déclinaisons (Gold Light, Gold Dark, Terra, Bronze) pour tous les éléments décoratifs ; réserver les bruns (`#3A2510`, `#5C3D1E`) uniquement au texte saisi dans les inputs
3. **Hiérarchie** : Cinzel Decorative pour le logo ; Cinzel pour boutons/CTA/navigation ; Uncial Antiqua pour labels formulaires ; Cormorant pour le contenu de lecture
4. **Espacement** : Respecter un rythme vertical régulier (multiples de 8px)
5. **Animations** : Subtiles et élégantes — runes en apparition/disparition, lueurs de torche, pas de rebonds
6. **Coins arrondis** : 16px pour les cartes, 8px pour les boutons et inputs, 2px pour les éléments de style médiéval
7. **Langue** : Français pour tout le contenu visible (`lang="fr"`)
8. **Décorations runiques** : Elder Futhark uniquement en tant que décoration visuelle, jamais comme texte lisible
